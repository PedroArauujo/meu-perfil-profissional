# Anotações da Semana 1

## O que aprendi

- HTML estrutura o conteúdo da página.
- CSS define a aparência da página.
- Tags semânticas ajudam a organizar melhor o código.
- Classes permitem estilizar partes específicas.
- Flexbox ajuda a organizar elementos lado a lado.
- Media queries ajudam a adaptar a página para celular.
- Git registra o histórico do projeto.
- GitHub funciona como vitrine do portfólio.

## Minhas maiores dificuldades

- Responsividade: entender quando usar media query, como testar em larguras diferentes e como evitar que cards, textos e botões fiquem apertados no celular.

## O que quero melhorar

- Continuar praticando responsividade nos próximos projetos.
